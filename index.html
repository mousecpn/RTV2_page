1KU Leuven

Abstract

We address the problem of (a) modeling the user's future behavior based on the past trajectory and contextual information, and (b) predicting the next desired action based on the current user's command through an uncertain user interface. We propose a probabilistic shared control framework for navigation, dubbed the Robot Trajectron V2 (RT-V2), which learns a decision-making distribution by imitation learning and imagined rollout reinforcement learning. RT-V2 realizes shared control by posteriorly estimating the desired action based on the current user's command. A sampling-based optimization method with constraints is employed to ensure the safety of RT-V2's posterior estimated actions, preventing collisions. Comprehensive experiments on trajectory prediction, navigation, and shared autonomy demonstrate that RT-V2 can accurately predict the user's intent and provide effortless assistance for users.

Visualization of failure trials of RT-V2. Both trajectory and entropy are plotted. The upper row for each subfigure visualizes the goals (green boxes), obstacles (pink boxes), distractors (white dash-line boxes), trajectories (orange lines), and user commands (light blue arrow). We visualize the user commands every 4 iterations for better visibility. The lower row for each subfigure visualizes the upper bound (Entropy UB) and lower bound (Entropy LB) of the entropy of the action-GMMs generated by RT-V2 for each iteration. Besides, the opacity of the trajectories represents the normalized entropy lower bound.
